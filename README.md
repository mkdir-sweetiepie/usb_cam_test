## Build & Usage
### build from source code
```bash
cd ~/${workspace_name}_ws/src
git clone https://github.com/mkdir-sweetiepie/usb_cam_test.git
cd ..
catkin_make
```

### How to use 
```
roslaunch usb_cam_test usb_cam_test.launch 
```
